# 🧠 쿠첸 AI 잡곡 추천 시스템  
## 추천 방식 & 테스트 가이드

---

## 1. 🎯 추천 알고리즘 개요

AI 잡곡 추천 엔진은 세 가지 주요 요소를 기반으로 동작합니다:

1. **건강 목표별 성분 기반 Z-score 스코어링**
2. **사용자 식감 선호(찰기 / 고슬 / 일반)** 가중치 적용
3. **기본 주곡(백미·현미 등) 비율 반영 후 최종 배합 계산**

---

## 2. 📊 건강 목표별 Z-score 기반 점수 계산

### ✔ 사용 데이터  
- 영양 데이터 매트릭스  
  https://docs.google.com/spreadsheets/d/1fZkKF3yEiMXFdGaujVuRzb1Bcgs8TuuXNN0OfJvbkxw/edit?gid=1515046506#gid=1515046506

### ✔ 계산 흐름
1. 건강 목표에 필요한 **핵심 영양 성분 리스트** 추출  
2. 잡곡별 해당 성분 값을 **Z-score 정규화**  
3. 성분별 Z-score 합산 → **해당 건강 목표 점수**  
4. 목표가 여러 개면 평균 또는 가중합 처리
